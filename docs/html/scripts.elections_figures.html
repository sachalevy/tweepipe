scripts.elections_figures.mongoDB.extract_header(list_of_queries: List[Dict[str, str]])¶

Creates the string header based on the list of queries. It always starts with start_date and end_date.

Example:

>>> list_of_queries = [
    {"name" : "total_tweets","query" : "query1"},
    {"name" : "total_tweets_with_hashtags", "query" : "query2"},
    {"name" : "fraud", "query": "query3"},
    {"name" : "fakenews", "query": "query4"}
]

>>> extract_header(list_of_queries)

‘start_date,end_date,total_tweets,total_tweets_with_hashtags,fraud,fakenews’

scripts.elections_figures.mongoDB.merge_two_dicts(x, y)¶

Helper function to merge dictionary x and y.

>>> x = {"tid": { "$gte": "1", "$lte": "2"}}
>>> y = {"json.entities.hashtags.text": { "$regex": "fraud", "$options": 'i'}}
>>> merge_two_dicts(x, y)
{'tid': {'$gte': '1', '$lte': '2'}, 'json.entities.hashtags.text': {
    '$regex': 'fraud', '$options': 'i'}}

>>> merge_two_dicts(x, {})
{'tid': {'$gte': '1', '$lte': '2'}}
